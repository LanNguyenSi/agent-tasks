# ADR 0002: Projects can be synced from GitHub

## Status
Accepted

## Context
Viele Projekte existieren faktisch bereits als Repository.

## Decision
Projekte können aus GitHub synchronisiert und automatisch provisioniert werden.

## Consequences
- Schnellere Nutzung
- Weniger Duplikate
- Zusätzliche Anforderungen an Sync-Status, Idempotenz und Fehlerbehandlung
