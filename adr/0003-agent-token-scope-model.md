# ADR 0003: Agent tokens use scopes

## Status
Accepted

## Context
Agenten sollen nicht unkontrolliert alle API-Aktionen ausführen können.

## Decision
Agent-Tokens enthalten explizite Scopes.

## Consequences
- Feinere Kontrolle
- Mehr Verwaltungsaufwand
- Bessere Sicherheits- und Governance-Eigenschaften
